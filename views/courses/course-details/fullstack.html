Detailed Course Syllabus

  • Module 1: Web Fundamentals
  • HTML5: Structure and Semantics
  • CSS3: Styling, Flexbox, and Grid
  • JavaScript: ES6+, DOM Manipulation
  • Module 2: Frontend Development
  • React: Components, Hooks, State Management
  • Redux for State Management
  • Tailwind CSS for Rapid Styling
  • Module 3: Backend Development
  • Node.js: Environment and Async Programming
  • Express.js: RESTful APIs and Middleware

  • Authentication: JWT, OAuth

  • Next.js: Server-Side Rendering, API Routes

  • CI/CD with GitHub Actions

  • Module 6: Projects and Portfolio
  • Project 1: Task Management App
  • Project 2: Social Media Platform
  • Capstone Project: Full Stack E-Commerce Platform
